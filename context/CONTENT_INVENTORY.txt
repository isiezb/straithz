Narrative Scene Types Coverage
-------------------------------
The narrative feed (js/narrative.js) supports 10 entry types:
  1. scene      — 2-4 sentence narrative prose (primary content)
  2. dialogue   — quoted NPC speech with speaker attribution and portrait
  3. alert      — urgent one-liner in red/yellow (critical/warning/good levels)
  4. stat       — metric change indicator (auto-preceded by context scene if orphaned)
  5. headline   — news ticker style, single line (also pushed to legacy SIM.headlines)
  6. command    — player action echo ("> DEPLOY ESCORT")
  7. consequence — action consequence narrative
  8. cable      — classified intel cable traffic (with portrait)
  9. advisor    — advisor reaction/aside (with portrait and speaker name)
  10. ambient   — ambient wire report
Portraits resolved via PORTRAIT_REGISTRY mapping character IDs, NPC names, and advisor titles to image paths.

AUDIO INVENTORY
========================================================================

- bgmusic.mp3: ~11 minute loop, triggered on title screen click, volume 0.3
- All other sounds are procedural via Web Audio API (SFX object in js/sound.js):
    - SFX.click() - UI interactions (square wave, 800->200Hz sweep)
    - SFX.chime() - good events, wins (3-tone sine arpeggio: 520/660/780Hz)
    - SFX.klaxon() - crisis events, interrupts (3-pulse sawtooth 440->380Hz)
    - SFX.phone() - decision events, diplomatic calls (dual-tone 440/480Hz ring)
    - SFX.crtBuzz() - CRT aesthetic (60Hz hum + noise, 0.3s)
    - SFX.error() - error states, bad outcomes (square wave 150->100Hz)
    - SFX.transition() - phase transitions, day changes (filtered noise sweep)
- No missing audio references (all procedural)
